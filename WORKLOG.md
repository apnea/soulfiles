## Worklog Guidelines

### Format

- Individual worklog files: `worklog/YYYY-MM-DD-HH-MM-SS-<title>.md`
- Index: This file (WORKLOG.md in root directory) links to all worklog files
- Legacy: Old single-file worklog archived in `worklog/archive/2026-03-27-legacy-worklog.md`

### Creating New Worklog Entry

1. Create new file in `worklog/` directory with format `YYYY-MM-DD-HH-MM-SS-<title>.md`
2. Add content
3. Update this index file (WORKLOG.md) with link to new entry
4. Commit changes with descriptive message

### Title Format

Use kebab-case for titles (lowercase with hyphens). Example: `session-state-schema-design` instead of `Session State Schema Design`.
